AI Agent Skills

I've built several custom AI agent skills for QE workflows — from ticket verification and pipeline analysis to performance benchmarking. Each skill is a codified, deterministic workflow executed by an AI assistant. Combined with 40+ tool integrations and productivity automations, I maintain an ecosystem of 50+ automated workflows.

QE-001

rpa-verification

End-to-end ticket verification: fetches Jira context, proposes 3-layer test plans (verification, regression, exploratory), executes them, and writes V&V results back to Jira.

QE-002

rpa-regression-testing

Executes the full hotfix regression catalog via Playwright CLI — covering RT execution, subtasks, credentials, support files, logs, and Operations Console.

QE-003

pipeline-failure-analysis

Analyzes GitLab CI test failures across parallel shards, classifies by error type/component, and determines if failures relate to a ticket's code changes.

QE-004

locust-prometheus-benchmark

Runs Locust performance tests against RPA services, collects p50/p95/p99 from Prometheus/Grafana, compares against baselines, and flags regressions >10% with annotated charts.

QE-005

weekly-quality-risk-report

Monday risk reports: critical incidents, CVEs over/approaching SLA, quality gates status, delivered via Google Chat.

QE-006

weekly-quality-summary

Friday quality metrics: bugs created/resolved, backlog trends, week-over-week deltas, delivered to the team automatically.
